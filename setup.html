How to use this tool

  1. For each flower or ribbon below — click its box or drag an image file onto it. PNG, JPG, and SVG all work.
  2. Skip any you want to leave as the existing pencil drawing.
  3. When you’re ready, click “Download data.js” at the bottom of the page.
  4. Go to your GitHub repository → click the existing data.js file → click the pencil (edit) icon → select all the text and delete it → paste in the contents of your downloaded file → click Commit changes.
  5. Your website updates within about 60 seconds. Everyone who visits will see the new images — permanently.
→ File size tip: Keep images under 500 kb each for fast loading. Free tool to compress: squoosh.app — just open it in your browser, drop your image, and download a smaller version.
