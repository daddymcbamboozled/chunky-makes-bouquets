Bloom Builder — Image Setup

One-time tool to swap flower images. Use this once, then upload the result to GitHub.

How this works

  1. For each flower, ribbon, or wrap below — click its box or drag an image onto it to set a picture.
  2. You can skip any you're happy leaving as-is. Ones you skip keep their existing drawing.
  3. When you're done, click Download data.js at the bottom.
  4. In your GitHub repository, delete the old data.js and upload this new one. That's it — your images are live for everyone.
ⓘ Images are embedded directly into the file, so nothing gets lost on refresh and no extra image files are needed. PNG, JPG, and SVG all work. Smaller files (under 500kb each) are recommended so the page loads fast.
✓ data.js downloaded!
Now go to your GitHub repository → find the existing data.js → click it → click the pencil icon → select "Delete file" and commit, then upload this new file. Your images will be live within 60 seconds.
